Project Status

Many people have expressed interest in doing a new version control system, and doing it right. Now, this project has been officially launched. Mailing lists, CVS repositories, and bugzilla are set up.

We are using CVS as an interim source code control system for our own project. It will be replaced once we have an initial working version of Inversion.
